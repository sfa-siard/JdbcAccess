JdbcAccess - Developers' Manual - Introduction

Introduction

JdbcAccess was developed by Enter AG for the Swiss Federal Archives as part of the SIARD (Software-Independent Archival of Relational Databases) project.

It serves the purpose of providing standardized façade in front of JDBC drivers of proprietary database management systems (DBMS). A homogeneous interface based on the JDBC 4.1 and the SQL:2008 standard can be presented to the user of the JDBC driver hding proprietary deviations from the standards behind this façade.

This document is the developers' manual of JdbcAccess.
